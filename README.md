# Akan Name Generator

## Description

This project incorperates javascript to assign akan names based on date of birth.
This web app calculates the day of the week you were born and assigns an Akan name based on gender.

## Author

Johncarlos Mwenda

## Setup

1. Clone repo
2. Open index.html write the code
3. Open style.css write a code to style the page
4. Use script.js to write the logic to **Retrive the user's input**, **Validate the input**, **Calculate the day of the week**, **Match the calculated day**, **Display the results**
5. Create a live site link using GitHub

## BDD (Behaviour-Driven Development)
Using **JAVASCRIPT** :
If a user enters a valid date and gender -> the correct Akan name is ouput and displayed
If the user enters an invalid day (e.g. 100) -> an error message is shown on the field prompting correct input
If the user enters an invalid month (e.g. 20) -> an error message is shown on the field prompting correct input
If the user doesnot select gender -> an error message is shown on the field prompting correct input
